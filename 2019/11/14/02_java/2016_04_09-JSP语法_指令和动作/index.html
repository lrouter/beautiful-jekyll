02_java/2016_04_09-JSP语法_指令和动作

JSP 语法

JSP文件，就是一个脚本文件，代码和html标签混合在一起组成的文件。

JSP文件，可以包含脚本代码段，声明段，纯粹的HTML标签段。

脚本代码段，是java代码，由<% %>包含。里面可以使用声明段定义的名字，也可以使用本文件其他代码段中定义的变量。

声明段，声明了变量和方法。作用域为本文件。

指令

由<%@ >包含的就是jsp指令，是在编译阶段起作用。用来设置整个JSP页面相关的属性，如网页的编码方式和脚本语言。

例如：
<%@ page language=”java” contentType=”text/html; charset=UTF-8”
pageEncoding=”UTF-8”%>
