一. 比较对象的两种方法

比较对象是否相等有两种方法：

  • 使用==运算符
  • 使用equals方法

1. ==运算符

==运算符，比较的是两个对象引用的值是否相等，也就是判断两个对象引用是否引用的是同一个对象。

2. equals方法

equals方法是Object类中的方法。Object是Java中所有类的父类，所以所有类都有这个方法。

Object类中的equals方法，比较的是两个对象引用的值是否相等，也就是判断两个对象引用是否引用的是同一个对象。和==运算符是一样的。

但是这并不是我们想要的。

重写equals方法

我们希望在使用equals方法时，比较的是两个对象的状态是否相等。
但是从Oject类继承的equals方法，比较的是两个对象引用的是否是同一个对象。
因此我们需要在创建自己类的时候，重写equals方法使得在比较对象时，比较的是对象的状态而不是对象的引用。

如果两个对象引用同一个对象，那么使用equals比较对象状态时肯定是判断相等，因为他们指向的是同一个对象啊！
